# POSTTRASH


## install:

`git clone https://github.com/MaximilianUE/POSTTRASH.git`

`cd ./web && npm install`


## usage: 

`npm run watch`

## to view postcss-styling correctly 

install Visual Studio Code Plugin: 'language-postcss' <br>
and add

``` 
"files.associations": {
    "*.css": "postcss"
}
```

to your workspace-settings
